Nicode Joomla Toolkit

Reusable utility library for Joomla 4, 5 and 6 extension development.

Overview

Nicode Joomla Toolkit is a lightweight PHP library designed to reduce repetitive code in Joomla extensions. It centralizes common tasks such as database queries, asset registration, request handling, escaping, UI helpers and file maintenance.

Compatibility

  • Joomla 4.x
  • Joomla 5.x
  • Joomla 6.x

Installation

Install the ZIP package via:

System → Install → Extensions

Included Modules

Module Class Purpose
Assets Nicode\Joomla\Toolkit\Assets Registers CSS/JS assets via WebAssetManager
Db Nicode\Joomla\Toolkit\Db\Database Query facade with bind support
Input Nicode\Joomla\Toolkit\Input\Request Safe GET/POST access
Date Nicode\Joomla\Toolkit\Date\DateText Human-readable date ranges
Html Nicode\Joomla\Toolkit\Html\Esc Safe HTML escaping helpers
Html Nicode\Joomla\Toolkit\Html\TableBuilder Generate accessible tables from arrays
Ui Nicode\Joomla\Toolkit\Ui\ProgressBar Accessible progress bar component
Fs Nicode\Joomla\Toolkit\Fs\Cleaner Old file cleanup utility
Util Nicode\Joomla\Toolkit\Util\Arr Array filtering helpers

Basic Usage

use Nicode\Joomla\Toolkit\Db\Database;

$value = Database::result('SELECT 1');

Web Assets

Installs media/lib_nicode_joomla_toolkit/joomla.asset.json with table and progress bar styles ready to use.
