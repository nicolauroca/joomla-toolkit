Nicode Joomla Toolkit

Reusable PHP utility library for Joomla 4, 5 and 6 extension development: database helpers, request/input handling, escaping, UI components, file maintenance and more.

Latest: v1.0.1 Joomla: 4.x · 5.x · 6.x License: GPL-2.0-or-later PSR-4

Overview

Nicode Joomla Toolkit is a lightweight PHP library designed to reduce repetitive code in Joomla extensions. It centralizes common tasks such as database queries, asset registration, request handling, escaping, UI helpers and file maintenance — keeping your extension code cleaner and easier to maintain.

Installation

Download the library ZIP from the Releases page and install it in Joomla: 

System → Install → Extensions → Upload Package File

After installation, the library is available to your extensions via its PSR-4 namespace.

Html Nicode\Joomla\Toolkit\Html\Esc Safe HTML escaping helpers
Html Nicode\Joomla\Toolkit\Html\TableBuilder Generate accessible tables from arrays

Fs Nicode\Joomla\Toolkit\Fs\Cleaner Old file cleanup utility
Util Nicode\Joomla\Toolkit\Util\Arr Array filtering helpers

Basic Usage

Example: run a simple database query using the toolkit facade: 

use Nicode\Joomla\Toolkit\Db\Database;

$value = Database::result('SELECT 1');

Use this inside your Joomla extension code where the Joomla framework is available.
