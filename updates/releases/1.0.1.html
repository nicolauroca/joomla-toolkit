Nicode Joomla Toolkit 1.0.1

Maintenance release of Nicode Joomla Toolkit, providing structural improvements and installation refinements for enhanced stability and compatibility across Joomla versions.

What's New in 1.0.1

  • Improved installation package structure
  • Manifest XML refinements for better compatibility
  • Minor internal optimizations
  • Confirmed compatibility with Joomla 4.x, 5.x and 6.x

Core Features

  • PSR-4 compliant architecture
  • Structured and reusable utility classes
  • Clean namespace organization
  • Joomla Update System support via <updateservers>

Compatibility

Fully compatible with:

  • Joomla 4.x
  • Joomla 5.x
  • Joomla 6.x
