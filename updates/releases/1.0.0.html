Nicode Joomla Toolkit 1.0.0

Initial public release of Nicode Joomla Toolkit, a lightweight and structured PHP toolkit designed to improve code quality, modularity and maintainability in Joomla 4, 5 and 6 extension development.

Highlights

  • PSR-4 compliant architecture
  • Structured and reusable utility classes
  • Clean namespace organization
  • Joomla Update System support via <updateservers>

Compatibility

Fully compatible with:

  • Joomla 4.x
  • Joomla 5.x
  • Joomla 6.x
